System

The basic building blocks used in CollectiveSpins.jl are, not surprisingly, spins. They are defined by their position and a frequency $\Delta$ describing a shift relative to the frequency of the rotating frame in use:

mutable struct Spin <: System
    position::Vector{Float64}
    delta::Float64
end

Defining the frequency is optional and is set to zero by default:

Spin([0,0,0]; delta=1)
Spin([0,0,0])

Combining many spins into one big system can be done by using the SpinCollection type. All contained spins must have the same polarization axis and decay rate $\gamma$:

mutable struct SpinCollection <: System
    spins::Vector{Spin}
    polarization::Vector{Float64}
    gamma::Float64
end

For convenience one can create a SpinCollection without explicitly constructing the single spins first::

SpinCollection([[0,0,0], [1,0,0]], [0,0,1]; gamma=2, delta=1)

Adding a cavity can be done with the CavityMode type:

mutable struct CavityMode <: System
    cutoff::Int
    delta::Float64
    eta::Float64
    kappa::Float64
end

which can be coupled to a spin collection with coupling strength g via the CavitySpinCollection type:

mutable struct CavitySpinCollection <: System
    cavity::CavityMode
    spincollection::SpinCollection
    g::Vector{Float64}
end
